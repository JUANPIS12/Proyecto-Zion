GUÍA RÁPIDA DE POSTMAN - API ZION
================================

Esta guía te ayuda a entender cómo realizar los llamados a la API.

--- 1. CÓMO CREAR UNA EMPRESA (POST) ---
Para crear un registro, se usa el método POST. 
URL: http://localhost:8080/api/empresas
Método: POST
Headers: Content-Type: application/json
Body (JSON):
{
  "nombre": "Zion Technology SAS",
  "ciudad": "Medellin",
  "direccion": "Calle 45 # 10-20",
  "contacto": "Juan Manuel",
  "estado": "ACTIVO",
  "sedeId": 1,
  "tecnologiaIds": [1, 2]
}

Explicación:
- "sedeId": Es el ID de la sede a la que pertenece la empresa.
- "tecnologiaIds": Es una lista de IDs de tecnologías que maneja la empresa.

--- 2. CÓMO ELIMINAR UNA EMPRESA (DELETE) ---
Para borrar un registro, se usa el método DELETE y se envía el ID en la URL.
URL: http://localhost:8080/api/empresas/1
Método: DELETE

Explicación:
- El número "1" al final de la URL es el ID de la empresa que quieres borrar. Si borras la empresa 5, la URL sería .../api/empresas/5.

--- 3. CÓMO ACTUALIZAR UNA EMPRESA (PUT) ---
Similar a crear, pero se envía el ID en la URL.
URL: http://localhost:8080/api/empresas/1
Método: PUT
Body (JSON): Envías todos los datos, incluso los que no cambian.

--- LISTADO GENERAL DE ENDPOINTS ---

- EMPRESAS: 
  GET /api/empresas (Listar todas)
  GET /api/empresas/{id} (Buscar una)
  POST /api/empresas (Crear)
  PUT /api/empresas/{id} (Actualizar)
  DELETE /api/empresas/{id} (Borrar)

- EQUIPOS:
  GET /api/equipos
  POST /api/equipos
  PUT /api/equipos/{id}
  DELETE /api/equipos/{id}
... (mismo patrón para todos)
