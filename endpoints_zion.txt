LISTADO DE ENDPOINTS - API ZION (CRUD COMPLETO)
==============================================

--- EMPRESAS ---
GET /api/empresas
Listar todas las empresas.

GET /api/empresas/{id}
Obtener una empresa por ID.
Params: id (Long)

POST /api/empresas
Crear una nueva empresa.
JSON Body:
{
  "nombre": "Empresa Ejemplo",
  "ciudad": "Ciudad",
  "direccion": "Calle 123",
  "contacto": "Juan Perez",
  "estado": "ACTIVO",
  "sedeId": 1,
  "tecnologiaIds": [1, 2]
}

PUT /api/empresas/{id}
Actualizar una empresa existente.
Params: id (Long)
JSON Body: (Mismo formato que POST)

DELETE /api/empresas/{id}
Eliminar una empresa.
Params: id (Long)
